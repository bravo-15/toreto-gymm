# TORETO GYM - Frontend React

Frontend completo en React + Vite para TORETO GYM.

## Incluye

- Landing page pública
- Login administrativo
- Dashboard
- Clientes
- Membresías
- Pagos
- Asistencias
- Layout administrativo con sidebar
- Datos de ejemplo listos para conectar al backend

## Instalar

```bash
npm install
```

## Ejecutar

```bash
npm run dev
```

## Rutas

- `/` página pública
- `/login` login admin
- `/admin/dashboard` panel principal
- `/admin/clientes` clientes
- `/admin/membresias` membresías
- `/admin/pagos` pagos
- `/admin/asistencias` asistencias

## Login demo

Puedes ingresar con cualquier correo y contraseña. Luego, cuando se haga el backend, se reemplaza por JWT real.
